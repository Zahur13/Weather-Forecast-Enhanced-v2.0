# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2024-01-XX

### Added

- Dynamic weather backgrounds (rain, snow, clouds, clear sky)
- Dark/Light mode toggle
- Geolocation support
- Hourly forecast (24 hours)
- Air Quality Index (AQI) display
- Favorites system
- Recent searches
- Search autocomplete
- Unit toggle (°C/°F)
- Weather alerts
- Secured API with backend proxy
- Rate limiting
- Responsive design improvements

### Changed

- Redesigned UI with glassmorphism
- Improved error handling
- Better mobile experience
- Enhanced animations

### Security

- API key now hidden in backend
- Added rate limiting
- CORS protection

## [1.0.0] - 2024-01-XX

### Added

- Initial release
- Basic weather display
- 5-day forecast
- Search functionality
